## Control Objective
Ensure that system access is authorized, appropriate, and periodically reviewed.

## Test Steps
1. Obtain a list of active users for in-scope systems
2. Select a sample of users based on risk
3. Verify documented approval for access
4. Confirm access aligns with job responsibilities
5. Validate evidence of periodic access review
6. Identify any exceptions or gaps

## Expected Evidence
- Access approval records
- User listings
- Access review attestations
- Issue remediation documentation
